Avance en algoritmo
Es el análisis masivo de datos excesivamente grandes para quee las aplicaciones de software de procesamiento de datos que tradicionalmente se venían usando sean capaces de capturar, tratar y poner en valor en un tiempo razonable. Un gran desarrollo del procesamiento paralelo, así como mayores frecuencias de CPU y finalmente el uso de potentes GPUS(graphics processing unit) para el procesamiento de datos. Los avances del algoritmo han pasado desde programas sencillos con condiciones y bucles hasta el denominado deep learning el cual utiliza millones de datos para realizar modelos predictivos.
